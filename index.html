Why this setup
Built for reliability
⚡
Instant Deploys
Every push to main branch triggers an automatic deployment — no waiting, no manual commands.
🔒
RHEL 9 Security
Running on Red Hat Enterprise Linux 9 with SELinux, firewalld, and enterprise-grade hardening.
☁️
AWS EC2
Hosted on Amazon EC2 for scalable, reliable cloud infrastructure with full control.
📋
Pipeline as Code
The entire deployment pipeline lives in a Jenkinsfile inside the repo — version controlled and auditable.
🌐
Apache Serving
Apache httpd serves the application on port 80, managed and restarted automatically by Jenkins.
🔄
Zero Downtime
Failed builds never reach production — the pipeline stops and Apache keeps serving the last good version.

Tech stack
Simple. Solid. Production-ready.

This entire CI/CD pipeline runs on a single AWS EC2 instance — no Kubernetes, no Docker, no over-engineering. Just the essentials that work.

  • Jenkins — pipeline orchestration
  • GitHub — source control + webhooks
  • Apache httpd — web server
  • AWS EC2 — cloud hosting
  • RHEL 9 — enterprise Linux OS
  • Jenkinsfile — pipeline as code
